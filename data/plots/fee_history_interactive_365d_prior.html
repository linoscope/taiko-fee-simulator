cost_wei(t_post) = l1GasUsed * (baseFee_t_post + priorityFee) + numBlobs * 131072 * blobBaseFee_t_post
FF_t = alpha_gas * (baseFee_(t-d_ff) + priorityFee) + alpha_blob * blobBaseFee_(t-d_ff)
de_t = epsilon_t - epsilon_(t-1), de_f_t = beta*de_f_(t-1) + (1-beta)*de_t
D_t = targetVault - vault_(t-d_fb), epsilon_t = D_t / targetVault, I_t = clamp(I_(t-1) + epsilon_t, Imin, Imax)
P_t = max(pFloor, Kp*epsilon_t*(maxFee-minFee)); fee_t = clamp(FF_t + P_t + Ki*I_t*(maxFee-minFee) + Kd*de_f_t*(maxFee-minFee), minFee, maxFee)
auto alpha uses BASE throughput: alpha_gas = l1GasUsed / l2GasPerProposal_base, alpha_blob = (numBlobs * 131072) / l2GasPerProposal_base

How Scoring Works

Goal: compare controller settings over the selected block range. Lower score is better.

total_badness = weighted_mean(health_badness, ux_badness)

health_badness = weighted_mean(dDraw, dUnder, dArea, dGap, dStreak)

  • dDraw: max under-target gap, normalized by target vault
  • dUnder: fraction of blocks with vault below target
  • dArea: deadbanded deficit area, normalized by target and range length
  • dGap: absolute final vault-target gap, normalized by target
  • dStreak: longest consecutive under-target run, normalized by range length

ux_badness = weighted_mean(uStd, uP95, uP99, uMax, uClamp, uLevel)

  • uStd: fee standard deviation, normalized by max fee
  • uP95/uP99: p95/p99 absolute fee step size, normalized by max fee
  • uMax: max absolute fee step, normalized by max fee
  • uClamp: fraction of blocks clamped at max fee
  • uLevel: average premium above break-even fee, normalized by break-even mean

Use the weight inputs to emphasize protocol health vs user UX, then click Score current range.
